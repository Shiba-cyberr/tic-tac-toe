# 🎮 Tic Tac Toe Game

A simple and fun Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**.
This project is beginner-friendly and demonstrates basic front-end web development concepts with interactive game logic.

---

## 🚀 Features

* ✅ Two-player gameplay
* ✅ Responsive design
* ✅ Win detection system
* ✅ Draw match detection
* ✅ Restart / New Game button
* ✅ Clean and simple UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 How to Play

* Player **X** starts first
* Players take turns marking boxes
* First player to align 3 symbols horizontally, vertically, or diagonally wins
* If all boxes are filled without a winner, the game ends in a draw
